## 安裝 keras
```
conda create –n NLP_course python=3.6
# NLP_course 是你的環境名稱
# python=X.X 是指定環境python版本，一般來說3.6 3.7

# 進入環境
conda activate NLP_course
# 如果忘記名稱
conda env list
# 退出環境
conda deactivate

conda install tensorflow

pip install keras
```
